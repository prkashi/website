Apriori algorithm

Data Mining

The purpose of the project is to design an Apriori algorithm in java in-order to form all the association rules possible for a transaction. These Association rules are stored in hash map in a key, value pair. The support and confidence values are user specified and the output will show all the association rules which satisfies the user requirement. All the transactions are stored in a text file and are read from java in-built functions.
◊ Java ◊ WEKA ◊ Eclipse

Chat Application

Java

A chat application which can send and receive messages from multiple clients and allow group chat and a shared white board. All the messages sent from a client is received by the server and are distributed to clients .
